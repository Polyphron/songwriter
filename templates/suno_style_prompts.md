# Suno Style Prompt Reference

Quick-reference for crafting Suno-ready style prompts. These go in square brackets
at the top of the lyric sheet.

---

## Format

```
[Genre, BPM, Key, Mood descriptors, Instrument/production elements, Vocal style]
```

## Examples by Genre

### Trap
```
[Dark trap, 135 BPM, minor key, heavy 808 slides, rolling hi-hats, atmospheric pads, aggressive male vocals]
[Melodic trap, 128 BPM, Eb minor, auto-tuned vocals, 808 bounce, lush reverb, emotional]
[Hard trap, 140 BPM, F# minor, distorted 808s, sharp snares, dark synth stabs, raw vocals]
```

### Drill
```
[UK drill, 140 BPM, G minor, sliding 808s, aggressive, dark piano riff, staccato flow]
[NY drill, 150 BPM, D minor, heavy bass, sample chops, menacing, raw vocals]
[Chicago drill, 145 BPM, A minor, dark, ominous, drill drums, auto-tune ad-libs]
```

### Grime
```
[Grime, 140 BPM, E minor, skippy drums, reese bass, MC energy, raw aggressive vocals]
[Dark grime, 142 BPM, C# minor, industrial synths, heavy sub, London MC flow]
```

### Boom Bap
```
[Boom bap, 92 BPM, Ab major, vinyl crackle, chopped soul sample, hard-hitting drums, lyrical]
[Underground boom bap, 88 BPM, D minor, dusty samples, jazz chords, introspective, raw mic]
```

### Melodic Rap
```
[Melodic rap, 125 BPM, Bb minor, auto-tuned vocals, piano chords, emotional 808s, spacious mix]
[Emo rap, 130 BPM, F minor, guitar loop, trap drums, vulnerable vocals, reverb-heavy]
```

### Cloud Rap
```
[Cloud rap, 70 BPM, C minor, lo-fi, ethereal pads, slow 808, dreamy, pitched vocals]
[Atmospheric rap, 75 BPM, G# minor, ambient textures, slow tempo, hypnotic, whispered vocals]
```

### Rage
```
[Rage, 155 BPM, E minor, distorted 808s, screaming ad-libs, aggressive, chaotic synths]
[Rage trap, 160 BPM, F minor, overdriven bass, triplet flow, high energy, mosh pit]
```

### Phonk
```
[Phonk, 130 BPM, A minor, cowbell, Memphis samples, dark, distorted, southern drawl]
[Drift phonk, 135 BPM, D minor, heavy bass, cowbell loops, aggressive, car culture]
```

---

## Key Elements to Include

| Element | Examples |
|---------|----------|
| **Genre** | trap, drill, grime, boom bap, phonk, rage, melodic rap |
| **BPM** | 70-170 depending on genre |
| **Key** | C minor, F# minor, Bb major, etc. |
| **Mood** | dark, aggressive, emotional, euphoric, menacing, introspective |
| **Instruments** | 808s, piano, guitar loop, synth stabs, soul sample, hi-hats |
| **Vocal style** | auto-tuned, raw, aggressive, mumble, melodic, spoken word |
| **Production** | reverb-heavy, lo-fi, clean mix, distorted, spacious, minimal |

---

## Tips

- Keep it under 30 words — Suno works better with dense, specific prompts
- Name specific instruments rather than saying "beats"
- Include the key if you know it (from audio analysis)
- Mood descriptors are more useful than genre labels alone
- "Male vocals" or "female vocals" helps Suno pick the right voice
